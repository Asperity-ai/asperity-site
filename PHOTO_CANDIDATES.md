# Hero background photo — candidates and selection

This file documents the image search that produced `hero-bg.jpg` plus runner-up
candidates if you want to swap. Sources: Pexels and Pixabay. (Openverse was also
queried; nothing in its top results beat what we found on the other two for the
black-and-white industrial / mechanical aesthetic, so no Openverse entries are
listed here.)

---

## Chosen image — `hero-bg.jpg`

- **Source:** Pexels
- **Photo ID:** 8294612
- **Photographer:** Pavel Danilyuk
- **Photographer page:** https://www.pexels.com/@pavel-danilyuk
- **Original photo page:** https://www.pexels.com/photo/close-up-shot-of-white-robot-toy-8294612/
- **Alt text (Pexels):** "Detailed close-up of a robot's mechanical components, emphasized by moody studio lighting."
- **Original dimensions:** 6016 x 4016 (downloaded and served at 1920 x 1280)
- **Average color (Pexels):** `#1B1F20` (near-black)
- **License:** [Pexels License](https://www.pexels.com/license/) — free for commercial and personal use, modification allowed, no attribution required (attribution appreciated).

### Why this one

It hits every brief criterion in one frame: a robotic arm / mechanical close-up
showing joints, fittings, and servo housings with moody studio lighting; already
nearly monochrome (avg color #1B1F20) so the B&W conversion needs almost
nothing; the entire left third is pure black negative space which is ideal for
a hero text overlay against the warm off-white site background and IBM Plex
typography. It is atmospheric, not promotional — there is no person, no blue
"AI" glow, no staged composition. It speaks directly to "AI-native robotics
deployment" without screaming it.

### Attribution

Pexels does not require attribution, but crediting is appreciated. Suggested
footer line if you want to credit:

> Hero photograph by Pavel Danilyuk on Pexels.

## Suggested footer / credits text

Pexels and Pixabay both allow commercial use without attribution. If you want
to credit anyway (good practice, helps photographers, and signals the site
isn't using paid stock), the smallest footer line below covers the chosen
image. If you swap to a runner-up, swap the line accordingly.

```
Hero photograph: Pavel Danilyuk / Pexels.
```

Or, if you also want to credit the source on the runners-up if they get used:

```
Imagery: Pavel Danilyuk (Pexels). Additional: Tima Miroshnichenko, cottonbro studio,
Rodrigo Pharazz, HONG SON, Sami TÜRK (Pexels); InWay, furbymama (Pixabay).
```

A standalone `/credits` page is overkill for a single hero image; a one-line
note in the existing footer is sufficient under both the Pexels and Pixabay
licenses.
